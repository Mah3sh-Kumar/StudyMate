# StudyMate - AI-Powered Study Assistant

**Transform your learning experience with intelligent study tools powered by AI**

StudyMate is a modern, cross-platform mobile application that leverages artificial intelligence to enhance the learning process. The app offers a suite of powerful tools designed to help students understand faster, revise smarter, and retain knowledge more effectively.

## Key Features

- **AI-Powered Content Summarization**: Instantly convert lengthy notes, textbooks, and study materials into concise, exam-ready summaries
- **Smart Quiz Generation**: Automatically create customized quizzes from your study material to test your knowledge
- **Interactive Flashcards**: Create and manage flashcards optimized for active recall and spaced repetition
- **AI Chat Assistant**: Get instant explanations and answers to your academic questions through contextual conversations
- **Hands-Free Studying**: Voice-controlled study sessions using speech recognition technology
- **Study Planning**: Generate personalized study schedules based on your goals and deadlines
- **Collaborative Learning**: Share study materials, quizzes, and flashcards with peers in real-time
- **Progress Tracking**: Monitor your learning journey with detailed analytics and performance metrics

## Tech Stack

- **Frontend**: React Native with Expo
- **Backend**: Supabase (PostgreSQL, Authentication, Realtime)
- **AI Services**: OpenAI API, GROQ API
- **Architecture**: Cross-platform mobile application with cloud synchronization

## Getting Started

1. Clone the repository
2. Install dependencies with `npm install`
3. Configure your API keys in the `.env` file
4. Run with `npx expo start`

## About

StudyMate transforms passive study into active learning by combining advanced AI models with an intuitive mobile interface. Whether you're preparing for exams, reviewing course material, or collaborating with classmates, StudyMate provides the tools to maximize your learning efficiency.
